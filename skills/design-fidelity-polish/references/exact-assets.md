# Exact Assets

Prefer the source-correct assets before hand-tuning CSS around approximations.

## Use Exact Assets When Acceptance Depends On Fidelity

- icons
- logos
- illustrations
- product imagery
- branded states

## Font Rule

If the user provides the exact font files, ingest them into the project.

If the exact font is unavailable:

- choose the closest stable fallback
- document the limitation clearly in the fidelity matrix

## Asset Stability Rule

Localize brittle remote assets into stable project paths when possible.
